Run Bad Apple-fier Locally

1. Install Python (PC only)

If you are on PC, install Python.

https://www.python.org/downloads/

2. Download it

You can download Bad Apple-fier from:

  • ZIP: download here
  • GitHub: here

3. Extract

Extract the files into a folder.

4. Run on PC

Open a terminal in the folder and type:

python -m http.server

Open this in your browser:

http://localhost:8000

5. Run on mobile

Install QuickEdit.

Extract the files, open index.html, and press Run.

Notes

Do not move files.
